INSTRUÇÕES PARA RODAR O PROGRAMA (Aluno.java)

1. Abra a pasta do projeto no seu editor Java (por exemplo: VS Code, Eclipse ou qualquer outro que tenha um botão 'Run').

2. Certifique-se de que o nome dos arquivos estão corretos. Foi gerado um arquivo para armazenar as informações sobre os alunos(Aluno.java) e um arquivo para rodar o código, onde será pedido as informações dos alunos(DadosAlunos.java)

3. No arquivo DadosAlunos.java, clique no botão "Run" (ou "Executar") no seu editor para rodar o programa.

4. O programa pedirá os dados de 10 alunos (nome, matrícula e média final).
   Digite cada informação e pressione Enter.

5. Após inserir os dados dos 10 alunos, o programa exibirá:
   - A lista de alunos aprovados (com média maior ou igual a 5.0)
   - A lista de alunos reprovados (com média menor que 5.0)

Não é necessário compilar manualmente com javac se você estiver usando um ambiente com botão de execução.
